## Para levantar el proyecto

-   Instalar las dependencias de php
    composer install
-   Modificar el archivo .env, configurar la conexion a la base de datos y crear una base de datos
-   Hacer las migraciones
    php artisan migrate --seed
-   Generar las llaves para la autenticacion (Passport)
    php artisan passport:install
